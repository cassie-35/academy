Curso: Machine Learning con Python

El curso de Machine Learning con Python te sumergirá en el apasionante mundo del aprendizaje automático, donde aprenderás a construir modelos predictivos utilizando una de las herramientas más populares en ciencia de datos: Python. Este curso está diseñado para proporcionarte una comprensión profunda de los conceptos clave del machine learning, desde los algoritmos supervisados hasta los no supervisados, el ajuste de hiperparámetros y la evaluación del rendimiento de los modelos. También aprenderás a utilizar bibliotecas populares como Scikit-learn, Pandas, NumPy y Matplotlib. Ya seas un principiante o tengas experiencia en programación, este curso te dará las habilidades prácticas necesarias para resolver problemas del mundo real utilizando machine learning.

¿Qué aprenderás?

  • Conceptos básicos de Machine Learning y su aplicación en la vida diaria.
  • Diferencia entre aprendizaje supervisado, no supervisado y por refuerzo.
  • Cómo manejar datos utilizando Pandas y NumPy.
  • Construcción de modelos de regresión lineal y logística.
  • Implementación de algoritmos de clasificación como KNN y Random Forest.
  • Métodos de clustering con K-means.
  • Evaluación del rendimiento del modelo mediante métricas como precisión, recall y F1-score.
  • Ajuste de hiperparámetros y técnicas de validación cruzada.
  • Uso de pipelines en Scikit-learn para automatizar procesos de machine learning.
  • Visualización de datos y resultados de modelos utilizando Matplotlib y Seaborn.

Contenido del Curso

El curso se divide en 5 partes principales para facilitar tu aprendizaje.

Tema 1: ¿Qué es Machine Learning y por qué es importante?
Tema 2: Instalación y configuración del entorno de desarrollo con Jupyter Notebook
Tema 3: Introducción a las bibliotecas de Python para machine learning
Tema 4: Primer proyecto: Crear un modelo básico de predicción
Tema 5: Importación, limpieza y manipulación de datos con Pandas
Tema 6: Tratamiento de datos faltantes y eliminación de outliers
Tema 7: Normalización y escalado de datos
Tema 8: Visualización de datos con Matplotlib y Seaborn
Tema 9: Práctica: Análisis exploratorio de datos en un conjunto real.
Tema 10: Regresión lineal: Conceptos y código en Python
Tema 11: Regresión logística y su aplicación en clasificación
Tema 12: Árboles de decisión y Random Forest
Tema 13: K-Nearest Neighbors (KNN)
Tema 14: Práctica: Comparación de diferentes algoritmos en un problema de clasificación
Tema 15: Introducción al clustering y algoritmos no supervisados
Tema 16: K-means clustering: Conceptos y aplicación práctica
Tema 17: Reducción de dimensionalidad con PCA (Análisis de Componentes Principales)
Tema 18: Detección de anomalías con algoritmos no supervisado
Tema 19: Proyecto: Implementar K-means en un conjunto de datos de clientes
Tema 20: Métricas de evaluación: Precisión, Recall, F1-score y ROC-AUC
Tema 21: Validación cruzada y técnicas para evitar el sobreajuste
Tema 22: Ajuste de hiperparámetros con GridSearchCV y RandomizedSearchCV
Tema 23: Automatización del flujo de trabajo con pipelines de Scikit-learn
Tema 24: Proyecto Final: Construir un modelo de machine learning de extremo a extremo
