Curso: Computación en la Nube con AWS

El curso Computación en la Nube con AWS: De Principiante a Experto es una guía completa para dominar los servicios en la nube de Amazon Web Services (AWS), la plataforma de computación en la nube más popular del mundo. Aprenderás los fundamentos de la nube, configuración de entornos, creación y administración de instancias EC2, manejo de almacenamiento S3 y seguridad en AWS. Además, se explorarán servicios avanzados como bases de datos RDS, redes VPC, y estrategias de alta disponibilidad y escalabilidad. Al final del curso, estarás preparado para implementar soluciones seguras y eficientes en la nube, e incluso prepararte para obtener certificaciones oficiales de AWS.

¿Qué aprenderás?

  • Conceptos básicos y modelos de computación en la nube.
  • Configuración y administración de instancias EC2.
  • Creación y gestión de almacenamiento en Amazon S3.
  • Configuración de bases de datos relacionales con Amazon RDS.
  • Gestión de usuarios y permisos con AWS IAM.
  • Configuración de redes virtuales con VPC (Virtual Private Cloud).
  • Implementación de alta disponibilidad y recuperación ante desastres.
  • Estrategias de escalado horizontal y vertical en AWS
  • Implementación de políticas de seguridad en AWS.
  • Mejores prácticas para la optimización de costos en AWS.

Contenido del Curso

El curso se divide en 5 partes principales para facilitar tu aprendizaje.

Tema 1: ¿Qué es la nube? Modelos y servicios principales
Tema 2: Ventajas de AWS frente a otras plataformas en la nube
Tema 3: Configuración de una cuenta de AWS y entorno inicial
Tema 4: Introducción a la consola de AWS y servicios más usados
Tema 5: Amazon S3: Creación y gestión de almacenamiento en la nube
Tema 6: Gestión de usuarios y permisos con AWS IAM
Tema 7: EC2: Creación y administración de instancias
Tema 8: Seguridad y control de acceso para instancias EC2
Tema 9: Creación de redes privadas con Amazon VPC
Tema 10: Proyecto: Implementación de una aplicación web simple en AWS
Tema 11: Amazon RDS: Configuración de bases de datos relacionales
Tema 12: Almacenamiento EBS: Volúmenes y snapshots
Tema 13: Backups y recuperación ante desastres en AWS
Tema 14: Escalado automático y balanceo de carga con AWS Elastic Load Balancer
Tema 15: CloudFront: Distribución de contenido globalmente
Tema 16: Estrategias de recuperación ante fallos y alta disponibilidad
Tema 17: Políticas de seguridad avanzadas en AWS
